Module Aef_lecture

module Aef_lecture: sig .. end
val lecture_car : Aef.etat -> char -> Aef.transition list -> Aef.etat option

La fonction lecture_car prend en entrée un état q, un caractère x et une liste de transitions les_transitions, et retourne l'état cible correspondant à la transition qui part de l'état q et qui est étiquetée par le caractère x

  • Author(s): Gilles, Marwan
  • Returns a type option
q : état
x : caractère
les_transitions : liste de transitions
val lecture_mot : Aef.etat -> char list -> Aef.transition list -> Aef.etat option

la fonction lecture_car vérifier s'il existe une transition pour le premier caractère du mot à lire et renvoie l'etat qui correspond à la transition d'une list de caractères

  • Author(s): Gilles, Marwan
  • Returns a type option
etat : état
mot : char list
les_transitions : liste de transitions
val accepter_mot : char list -> Aef.aef -> bool

La fonction accepter_mot utilise la fonction lecture_mot pour parcourir le mot dans l'AEF et renvoie true si le dernier état atteint après avoir parcouru le mot appartient à l'ensemble des états finaux de l'AEF

  • Author(s): Gilles , Marwan
  • Returns a boolean
